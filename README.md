# Cleaning Nashville Housing Data in SQL Queries
This project aims to clean the Nashville Housing dataset using SQL queries. The dataset contains information about properties, such as their addresses, prices, and property characteristics. The data is cleaned using SQL queries to handle missing values, remove duplicates, and standardize the format of the data. The cleaned data is then used for further analysis and visualization.

## Conclusion
By cleaning the Nashville housing dataset using SQL queries, the project aims to improve the quality of the data and make it more suitable for further analysis and visualization. The cleaned dataset can be used to gain insights into the Nashville housing market and inform real estate decisions.
